PRIORITIES FOR LIFE LANDING PAGE

FILES
- index.html: Complete preview page.
- body-snippet.html: Paste-ready block containing all HTML, CSS, and JavaScript.
- assets/: Images used by the page.

DEPLOYMENT
1. Upload the assets folder to the same directory as the landing page.
2. Paste body-snippet.html inside the destination site's body/content editor.
3. If the platform changes the asset location, replace every assets/... URL with an absolute URL.
4. Calculator values are grouped near the start of the JavaScript in the programData object.

NOTES
- CSS is scoped beneath #pfl-page to reduce conflicts with dealership-site styling.
- No external libraries or fonts are required.
- Lexus and Mitsubishi values can be toggled.
- Savings are proportionally estimated from the published 10-year values.
